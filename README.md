# Azure AZ-104 Hands-On Labs
This repository contains hands-on Azure labs I built after passing the AZ-104 exam.
The goal is to demonstrate real-world Azure administration skills used in enterprise
and Norwegian SMB/municipality environments.

## Labs Included
- Entra ID & RBAC
- Hybrid Identity (On-Prem AD + Azure)
- Azure Networking (Hub & Spoke)
- High Availability Web Application
- Backup & Disaster Recovery
- Monitoring & Cost Management

Each lab includes:
- Architecture diagram
- Step-by-step setup
- Screenshots
- Key learnings
